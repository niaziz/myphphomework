PHP Variable & scope

Creating or declearing variables in PHP

- In PHP variables starts with $, followed by the name of variable.

‹?php
$txt = "Hello world!";
$primaryVarible = 7;
$variable= 0.9;
?> - After the excution of the above statment the "Hello world!" will be stored in $txt variable.
- Think of variables as containers for stuff.
- Variables can have short names such as $X or $Y or more descriptive names such as $name,$age, $BirthDate...


Rules for PHP variables
* A variable starts with the $ sign, followed by the name of the variable.
* A variable name must start with a letter or the underscore character
* variable name cannot start with a number
* A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
* Variable names are case-sensitive ($age and $AGE are two different variables)
